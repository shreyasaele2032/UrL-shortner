# 🔗 URL Shortener

A simple and efficient URL Shortener application that allows users to convert long URLs into short, shareable links. The application provides quick redirection, link validation, and an intuitive user interface for managing shortened URLs.

---

## 🚀 Features

- 🔗 Shorten long URLs instantly
- 🌐 Redirect shortened links to the original URL
- ✅ URL validation before shortening
- 📋 One-click copy to clipboard
- ⚡ Fast and responsive interface
- 📱 Mobile-friendly design
- ❌ Error handling for invalid URLs

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/url-shortener.git
```

2. Navigate to the project

```bash
cd url-shortener
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
